# Advar Atlas 4.1.1

This release corrects the phone presentation and terminology. Phones now use a landscape-only Atlas view so the complete monochord is visible. Safe-area padding prevents overlap with the system status bar. The opening screen is shorter, the previous-session and reset controls are removed, public test panels are removed, Persian Music Scales is translated as «گام‌های موسیقی ایرانی», and the brand mark is simplified to the manuscript-inspired circular diagram.
