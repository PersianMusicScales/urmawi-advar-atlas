# Al-Urmawī Advar Atlas 4.3.0

This release is the final interaction and responsive-layout repair.

## Corrected functionality

- Opening-screen buttons initialize without JavaScript exceptions.
- Combined, Theory, Instrument, and Focus controls work independently on desktop.
- Fullscreen remains available on desktop and is hidden on phones.
- All six learning sections are directly visible in phone landscape navigation.
- Each phone navigation item opens its corresponding content panel.

## Phone presentation

- Portrait phone use displays the landscape-orientation guidance.
- Landscape phone use fits the full theoretical monochord across the screen.
- The illustrative instrument remains hidden by default on phones.
- Safe-area spacing protects the header and navigation from system UI.

## Language and terminology

- English, Persian, Arabic, and Turkish dynamic content refreshes after every language change.
- Persian creator wording uses «محقق».
- Persian Music Scales is rendered as «گام‌های موسیقی ایرانی».
- Public product terminology consistently uses Atlas.

## Interface cleanup

- Removed font-size and contrast controls.
- Removed obsolete session-continuation, reset, and public test displays.
- Centered and aligned the circle-only brand mark.
