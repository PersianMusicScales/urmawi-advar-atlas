# Al-Urmawī Advar Atlas 4.0.0

This release transforms the Atlas into an official Persian Music Scales progressive web application.

## Highlights

- Manuscript-inspired official icon, color system, launch screen, and social preview
- Installable and offline-capable PWA
- True responsive desktop, tablet, and mobile layouts
- Theory-first phone experience with optional instrument view
- Mobile bottom navigation, touch controls, long-press listening, lesson swiping, and theory zoom
- Persistent lesson progress, quiz score, selected pitch, view, audio, and accessibility preferences
- English, Persian, Arabic, and Turkish coverage for all new PWA controls
- Dedicated Google Sites embed mode
- GitHub Pages, validation, Lighthouse, and release workflows
